**ENEDAR - Frontend library.**

_**Quick start**_

Use npm or yarn 
```bash
yarn add enedar
# or 
npm install enedar
```

Add to you project `enedar.min.css` or `enedar.scss`

Latest stable version `1.1.0 - "Sputnik"`

Latest non-tested version `1.1.3`

**Tips**

All vectors default rendering 200px * 200px
